Modeling ground motion

Ground motion model based on 2D spectrum P(w,k) is now systematically supported in the LIAR (LInear Accelerator Reseach code). Please refer to LIAR web pages for the soures and description.

An older version of the stand-alone program is also available here: version Oct 2000 .
