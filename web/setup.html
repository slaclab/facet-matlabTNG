The LUCRETIA Project

Setup Instructions


  • Download Lucretia package to a desired installation folder
  • Make a working directory
  • Add the Lucretia shared objects to your dynamic library search path (see download section for more details)
  • Add required Lucretia directories to your Matlab search path (make a startup.m file with required "addpath" commands or use Matlab's pathtool)
        e.g. Lucretia/src/Tracking for TrackThru command
        e.g. Lucretia/src/utils for various utility commands etc...
    • You can use the LucretiaInit.m function provided in the root Lucretia directory to map all Lucretia directories.
  • To use the GEANT4 interface use the LucretiaInit.m function with the 'cpu-g4' calling argument, this will automatically select the GEANT4-built binaries for use.
  • Try out some of the examples in Lucretia/src/Examples.
