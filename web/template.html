The LUCRETIA Project

Examples

Often the best way to learn a new simulation code is to start from an example or two; this will also help to indicate whether the code is working or not.  At the moment there are 2 Lucretia examples available, and they are included in the Lucretia Zip files (see downloads), or the source code of the examples can be browsed (follow the links in the example names).

ILC BC:  A "straw man" single-stage BC + main linac.  After instantiation the example will generate Twiss parameters, track bunches on-axis and with a 1 sigma_y initial offset, examine the longitudinal phase space variation as the BC RF phase is varied, and do some simple studies of transverse sensitivities to misalignments.  Finally, a bunch train (with 1 ray/bunch) is tracked through the linac with LRWFs, including rotations and mode-splittings.
