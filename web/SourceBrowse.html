Browse the Source!

The source for all Lucretia files, including the web documentation, is under CVS control.  The best way to browse the source is therefore to use the CVSWEB interface to the Lucretia CVS archive.

Users at SLAC may want to run Lucretia off of the "official," "production" sources.  Unfortunately, due to CVS name- and source-mangling, it's not possible to run directly off the copies stored in CVS.  For this reason, we have a "checked-out" version which we store locally here (AFS location:  /afs/slac/g/ilc/codes/Lucretia/src).  The checked-out version should be an exact image of the CVS version.  If you find that this is not the case, please immediately chew out the Lucretia team and demand that the divergence in the two versions be fixed forthwith.
