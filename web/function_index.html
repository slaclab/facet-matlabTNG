Script, Function and Class Documentation
A wide variety of scripts, functions and classes have been written for use in Lucretia, in a number of categories.  Note that all functions are somewhat self-documented, and the documentation can be seen via the Matlab help operation:  type "help functionname" at the Matlab prompt. Additionally, classes have further documentation which can be accessed by typing "doc classname". This gives more documentation on the public methods and properties of the class.
