Other Useful Stuff:  Documentation

The main documentation for Lucretia is this website.  All of the functions, data structures, etc., are documented herein.  In addition, all of the Lucretia functions can make use of Matlab's help command.  Typing, "help (functionname)" at the Matlab prompt will cause a description of the function, including syntax, to appear.

At the moment there is not a printed manual as such for Lucretia.  So far, I think that the combination of command-line help and online documentation is more useful.  However, there are times when the network is not available but help is required.  In order to address that sort of issue, the full contents of the Lucretia website have been put into a Zip'ed archive, which is available from the Downloads web page.
